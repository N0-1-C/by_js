# 文件加密解密工具 - 客户端版本 1

这是一个基于JavaScript的文件加密解密工具，所有数据处理都在用户的浏览器中完成，无需上传文件到任何服务器。

## 功能特点

- **完全客户端处理**：所有加密、解密操作都在浏览器中完成，文件不会上传到任何服务器
- **安全加密**：使用Web Crypto API的AES-GCM算法进行加密
- **大文件支持**：采用分块处理技术，支持大文件的加密解密
- **用户友好界面**：直观的标签式界面，支持生成密钥、加密文件、解密文件
- **灵活的密钥管理**：支持密码和密钥文件两种方式进行加密解密
- **双重模式支持**：用户可以选择使用密码或密钥文件进行加密解密

## 使用方法

1. **生成密钥**：点击"生成密钥"按钮创建加密/解密所需的密钥文件（向后兼容）
2. **加密文件**：
   - 密码模式：选择"使用密码加密"，输入密码，选择文件进行加密
   - 密钥文件模式：选择"使用密钥文件加密"，选择密钥文件和文件进行加密
3. **解密文件**：
   - 密码模式：选择"使用密码解密"，输入密码，选择.pv加密文件进行解密
   - 密钥文件模式：选择"使用密钥文件解密"，选择密钥文件和.pv加密文件进行解密

## 技术实现

- **加密算法**：AES-GCM 256位对称加密
- **分块大小**：64KB（与Python版本保持一致）
- **密钥管理**：支持密码派生密钥（使用PBKDF2）和密钥文件两种方式
- **文件处理**：使用FileReader API进行分块读取和处理
- **安全派生**：使用PBKDF2算法和100,000次迭代确保密码安全

## 安全说明

- 所有文件处理都在您的浏览器中进行，不会上传到任何服务器
- 使用PBKDF2算法从密码派生加密密钥，确保安全性
- 密码通过高强度迭代算法转换为加密密钥，防止暴力破解
- 请记住您设置的密码或妥善保管密钥文件，丢失将无法解密文件
- 保留了密钥文件功能以确保向后兼容性

## 兼容性

- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 需要支持Web Crypto API
- 需要支持FileReader API
- 需要支持PBKDF2算法
